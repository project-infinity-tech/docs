This page is an honest snapshot of what Scram doesn't support today. If you're hitting any of these as a blocker, post in the [Builders group on Slack](REPLACE_WITH_SLACK_LINK) so we can understand the demand and see what we can do.

## CSV upload

Scram doesn't currently support uploading and parsing CSV files as a data source.

This applies whenever you want users to bulk-import data via a CSV, populate a table from an uploaded spreadsheet, or accept CSVs as a workflow input.

If this is a blocker for what you're building, post in the [Builders group on Slack](REPLACE_WITH_SLACK_LINK) so we can see what we can do.

## Realtime updates

Scram doesn't currently support server-sent events or WebSocket connections for pushing updates from the server to the browser.

This applies whenever you're building chat-style apps, live collaborative editing, presence indicators, in-app notifications, or anything else that needs the server to push updates to the browser as they happen.

If this is a blocker for what you're building, post in the [Builders group on Slack](REPLACE_WITH_SLACK_LINK) so we can see what we can do.
