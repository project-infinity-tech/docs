title: "Scrambot"
description: "System-level functions and settings accessed via the Scrambot menu in Scram."
---

Scrambot is the system-level menu in the Scram editor — think of it as the hamburger menu for platform-wide functions. Click the Scrambot icon in the Top Navigation to open it.

---

## Useful stuff

**Go to Workspace** — returns you to your project list, leaving the current editor.

**Publish** — takes you directly to the Deployments page to publish a new version to Live.

**Keyboard Shortcuts** — opens a reference of all available keyboard shortcuts in the editor. See the [Keyboard Shortcuts](/reference/keyboard-shortcuts) reference for the full list.

**Log Out** — logs you out of Scram.

---

## Debug items

Debug options are mostly provided for use when troubleshooting with Scram support. One exception worth knowing about:

**Reset Pane Layout** — resets the editor panes back to their default layout. Useful if you've accidentally resized or moved panels and can't get back to a sensible state.

---

## Feature flags

Feature flags enable experimental or optional behaviours. Use these with care — they may not be fully stable.

| Flag | Description |
|---|---|
| AI Conversation Clearing | Automatically summarises AI chat history to reduce context size when conversations get long |
| Detect AI Tab Throttling | Automatically pauses the AI if the browser tab becomes inactive, preventing throttling issues |
| Mixpanel Analytics | Allows Scram's own analytics to run in your session |
| OpenRouter Integration | Experimental access to OpenRouter as an alternative AI provider |
