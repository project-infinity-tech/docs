---
title: "Side Navigation"
description: "The context-sensitive panel that runs down the left side of the Scram editor."
---

The side navigation runs down the left edge of the editor and is always visible. Its contents change depending on which view you're currently in — it shows only the tools relevant to what you're working on.

---

## Canvas mode

When working on a page in the Canvas, the side navigation contains three sections:

**AI Chat** — the Scram AI assistant. Ask it to build components, explain concepts, or help you work through a problem. Available in all views.

**Page Structure** — a tree view of every component on the current page, showing the hierarchy of containers and elements. Use it to select, reorder, and navigate between components — particularly useful on complex pages where clicking on the canvas is difficult.

**Component Editor** — appears when a component is selected. Contains three tabs:

- **Properties** — configure what the component displays and how it behaves.
- **Styles** — control the visual appearance of the component — colours, spacing, typography, and layout.
- **Logic** — define the events and workflows attached to this component, such as what happens when it is clicked or when its value changes.

---

## Workflow Editor mode

When working in the Workflow Editor, the side navigation contains:

**AI Chat** — the Scram AI assistant, as above.

**Workflow Config** — configuration options for the current workflow, including its trigger, settings, and metadata.

---

## Resource View mode

When working in the Resource View, the side navigation contains:

**AI Chat** — the Scram AI assistant, as above. The Resource View tools are accessed directly within the main area rather than through the side navigation.
