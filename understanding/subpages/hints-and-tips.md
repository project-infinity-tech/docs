## Handling long running processes

If you often change tabs and leave Scram’s AI working through a task list we recommend turning off your browser’s default Tab throttling behaviour for the Scram site so that it can continue to run uninterrupted at top speed. The process for doing this varies by the browser that you use. Here are some examples:

- **In Google Chrome**
    - Open the performance settings from chrome://settings/performance and find this option:
    
    ![image.png](image%2015.png)
    
    - Click ‘Add’. If you already have Scram open in Chrome you can add it to the list by choosing it from the checklist presented. Otherwise you can use the ‘Add sites manually’ option.
        
        ![image.png](image%2016.png)
        
- In Microsoft Edge
    - Open the performance settings from edge://settings/system/managePerformance
    - Add the site to the always keep these sties active. https://app.projectinfinity.tech
    - Also consider unchecking the tab under Memory  - “Automatically put tabs to sleep”
    
    ![image.png](image%2017.png)
