title: "Keyboard Shortcuts"
description: "A reference of all keyboard shortcuts available in the Scram editor."
---

## Everywhere

| Shortcut | Action |
|---|---|
| `?` | Show keyboard shortcuts |
| `Ctrl + Z` | Undo last action |
| `Ctrl + Y` | Redo last action |
| `Alt + Right-click` | Browser right-click menu |

---

## Page and Component Editors

| Shortcut | Action |
|---|---|
| `Ctrl + C` | Copy selected component |
| `Ctrl + X` | Cut selected component |
| `Ctrl + V` | Paste component |
| `Ctrl + .` | Add new component |
| `Delete` | Delete selected component |
| `Escape` | Unselect selected component |
| `Alt` | Hide selection outlines and controls |

---

## Inputs

| Shortcut | Action |
|---|---|
| `Ctrl + Enter` | Toggle expression mode |
