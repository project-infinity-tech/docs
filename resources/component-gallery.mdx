The Component Gallery is a visual showcase of the components available in your project — both the platform components provided by Scram and any custom components you've built yourself.

It's a reference view rather than an editing tool. To add a component to a page, use the **Insert Component** option from the Canvas or the Page Structure panel.

---

## Adding your own components

The Component Gallery includes an option to create a new custom component. For a full guide on building your own components, see [Building Your Own Components](/understanding/subpages/building-your-own-components).

---

## Platform Components version

The Component Gallery also shows the current version of the Scram platform components. From here you can upgrade or downgrade the component version your project uses.

<Warning>
  Downgrading the platform components version is not advised without first contacting Scram support — it can introduce unexpected behaviour in your app.
</Warning>
